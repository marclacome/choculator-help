Collection Filter Buttons (admin - settings)

There are sections for each individual collection which has filter buttons

1

Create the collections which you will link to

Create collections as you normally would in Shopify admin.

For each collection you can create up to 20 "filtered collections" - ie. Ghirardelli plus up to 20 Ghirardelli filtered collections (ghirardelli bars, ghirardelli squares etc), Lindt-Chocolate plus up to 20 Lindt-Chocolate filtered collections (lindt bars, lindt uk etc) etc

VERY IMPORTANT! Please read the following instructions about naming conventions for filtered collections first.

The url for each filter must match a naming convention -

  • Ghirardelli
  • Sub collection url must start with "ghirardelli-chocolate-"
  • Best of the rest
  • Sub collection url must start with "best-of-the-rest-"
  • Lindt Chocolate
  • Sub collection url must start with "lindt-chocolate-"
  • Gift collections
  • Sub collection url must start with "chocolate-gifts-"
  • Chocolate Bars
  • Sub collection url must start with "chocolate-bars-"

To set the url, when you create a collection give it a title which starts with the above url, replacing '-' with a space. ie to add an easter collection to Gift Collections, you would call the collection Chocolate Gifts Easter, giving a url of chocolate-gifts-easter

If you've already created the collection you can change the url on the collection page - scroll to the bottom of the page
